Likelihood and Priors
======================

Likelihood evaluation and prior distribution modules.

Overview
--------

These modules provide the likelihood functions and prior distributions used by the
:doc:`integrators <../integrators/index>` for parameter inference. They handle
the computation of the likelihood of gravitational wave signals given detector data.

.. seealso::

   - :doc:`../integrators/index` - MC integration modules that use these likelihoods
   - :doc:`../physics/index` - Physical models and utilities
