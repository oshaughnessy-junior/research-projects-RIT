Core Executables Overview
=========================

The following executables form the core user interface for RIFT:

1. **util_RIFT_pseudo_pipe.py** - Primary user interface for setting up RIFT analyses
2. **create_event_parameter_pipeline_BasicIteration** - Pipeline writer for condor DAG workflows
